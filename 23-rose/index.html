Chart Configuration 
const config = {
    data,
    xField: 'type',
    yField: 'value',
    seriesField: 'type', // 使用seriesField确保每个分类使用不同颜色
    radius: 0.9,
    color: ['#5B8FF9', '#5AD8A6', '#F6BD16', '#945FB9', '#E86452'],
    
    // 玫瑰图特有：用半径表示数值
    sectorStyle: {
        stroke: '#fff',
        lineWidth: 2,
    },
    
    label: {
        offset: -15,
        content: ({ percent }) => `${(percent * 100).toFixed(0)}%`,
        style: { fill: '#fff', fontSize: 11 },
    },
    
    legend: false,
    interactions: [{ type: 'element-active' }],
};
玫瑰图规范
参数值说明
radius 0.9 外半径比例
xField string 分类字段
yField string 数值字段（决定半径长度）
sectorStyle.stroke #fff 扇形边框颜色
sectorStyle.lineWidth 2 扇形边框宽度
label.offset -15 标签偏移距离
label.style.fontSize 11px 标签字号
使用建议
建议说明
数据差异适合展示有明显差异的分类数据，半径差异更直观
分类数量建议 4-8 个分类，过多会显得拥挤
适用场景适合展示产品销量、地区分布等有明显差异的占比数据
