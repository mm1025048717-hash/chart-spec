⏱️ 指标组件

仪表盘

展示指标的完成进度或当前状态，直观展示目标达成情况

@ant-design/charts Gauge 进度展示
常见变体
基础仪表盘 Basic
当前达成率75%
多色仪表盘 Multi Color
当前达成率65%
刻度仪表盘 With Ticks
当前达成率82%
核心配置参数
参数值说明
percentnumber (0-1)当前进度百分比
range.colorstring | string[]进度条颜色
indicatorobject | null指针配置
statisticobject中心统计文本
代码示例
import { Gauge } from '@ant-design/charts'; const config = { percent: 0.75, range: { color: '#5B8FF9' }, indicator: { pointer: { style: { stroke: '#D0D0D0' } }, pin: { style: { stroke: '#D0D0D0' } }, }, statistic: { content: { formatter: ({ percent }) => `达成率: \${(percent * 100).toFixed(0)}%`, }, }, };
