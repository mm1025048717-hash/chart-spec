方案 02 · 动态平衡

中等文字倾斜显示

当文字长度处于 5-8 字符之间时，45° 倾斜是兼顾可读性与空间效率的黄金分割点。

图表预览
设计尺寸规范
文字长度 (Limit)5-8 字符
数据项 (Count)≤ 12 项
轴标签字号 (Size)11px
轴标签颜色 (Color)#86868B
文字旋转 (Rotate)-45° (-π/4)
文字对齐 (Align)End (Right)
柱体宽度 (Ratio)0.5
顶部圆角 (Radius)4px
应用准则
✓
对齐逻辑：必须设置 `textAlign: 'end'`。这能保证倾斜后的文字末尾精准指向刻度线。
✓
阅读流向：45° 倾斜符合从左往右、从上往下的阅读习惯，是最推荐的折中方案。
✓
间距补偿：倾斜会增加底部空间占用，建议 X 轴底部 Margin 至少保留 60px。
核心配置代码
xAxis: {
  label: {
    autoRotate: false,
    rotate: -Math.PI / 4,
    style: {
      fontSize: 11,
      textAlign: 'end'
    }
  }
}
