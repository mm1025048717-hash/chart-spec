方案 03 · 紧凑策略

长文字垂直显示

面对超长文本，垂直排布是唯一的解法，它能在保证文字完整性的同时，最大化利用垂直空间。

图表预览
设计尺寸规范
文字长度 (Limit)> 8 字符
数据项 (Count)可支持 ≥ 10 项
轴标签字号 (Size)11px
文字颜色 (Color)#86868B
旋转角度 (Rotate)-90°
对齐方式 (Align)Right
基准线 (Baseline)Middle
柱体宽度 (Ratio)0.4
应用准则
!
空间预留：由于文字垂直显示，图表底部需要预留较大的 Margin（通常 ≥ 120px），以防文字被截断。
!
对齐关键：`textAlign` 必须设为 `right`，使长短不一的文字末端均对齐坐标轴线，视觉整齐。
!
阅读效率：仅在必要时使用，因为垂直阅读效率低于水平或倾斜阅读。
核心配置代码
xAxis: {
  label: {
    autoRotate: false,
    rotate: -Math.PI / 2,
    style: {
      fontSize: 11,
      textAlign: 'right',
      textBaseline: 'middle'
    }
  }
}
