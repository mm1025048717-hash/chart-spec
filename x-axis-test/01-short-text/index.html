方案 01 · 极简策略

短文字水平显示

当文本内容极为精炼时，保持水平显示能提供最直观、自然的阅读动线。

图表预览
设计尺寸规范
文字长度 (Limit)≤ 4 字符
数据项 (Count)≤ 8 项
轴标签字号 (Size)12px
轴标签颜色 (Color)#86868B
文字旋转 (Rotate)0°
文字对齐 (Align)Center
柱体宽度 (Ratio)0.5
顶部圆角 (Radius)4px
应用准则
✓
视觉平衡：确保柱体间的空隙（Gaps）与柱体本身宽度成正比，避免显得稀疏或拥挤。
✓
对齐方式：文字必须与柱体中心严格对齐，形成垂直向上的视觉引导。
✓
负空间：保持图表上方至少有 20px 的留白，用于显示数据标签。
核心配置代码
xAxis: {
  label: {
    autoRotate: false,
    rotate: 0,
    style: {
      fontSize: 12,
      textAlign: 'center'
    }
  }
}
