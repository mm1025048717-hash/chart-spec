核心配置参数
参数 值 说明
angleField string 扇形角度映射字段（数值）
colorField string 颜色映射字段（分类）
radius 0.8 饼图半径
innerRadius 0 或 0.6 内径（0为饼图，>0为环图）
代码示例
import { Pie } from '@ant-design/charts'; const data = [ { type: '线上直销', value: 4500 }, { type: '线下门店', value: 3200 }, { type: '分销渠道', value: 2800 }, ]; const config = { data, angleField: 'value', // 数值字段 colorField: 'type', // 分类字段 radius: 0.8, label: { type: 'outer', content: '{name} {percentage}', }, legend: { position: 'right' }, }; return <Pie {...config} />;

使用建议
建议 说明
分类数量 建议不超过7个，过多时考虑合并或使用其他图表
数据差异 各部分差异不宜过大，否则小部分难以识别
环形图 中心可放置关键指标，增强信息传达
