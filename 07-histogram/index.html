直方图规范
参数值说明
binField string 分箱字段（数值型）
binWidth number 分箱宽度（区间大小）
binNumber number 分箱数量（与binWidth二选一）
columnWidthRatio 0.5 柱体宽度占比
columnStyle.radius [4,4,0,0] 柱体圆角（顶部圆角）
color #5AD8A6 柱体颜色
使用建议
建议说明
分箱宽度建议根据数据范围选择合适的分箱宽度，确保分布清晰可见
分箱数量建议 10-20 个分箱，过少或过多都影响分布识别
适用场景适合展示连续数据的分布情况，如年龄分布、收入分布等
代码示例
import { Histogram } from '@ant-design/charts'; const config = { data, binField: 'age', binWidth: 5, // 每5岁一组 color: '#5AD8A6', columnStyle: { radius: [4, 4, 0, 0] }, };
