Chart Configuration 
const config = {
    data,
    xField: 'type',
    yField: 'value',
    padding: [40, 24, 60, 60],
    
    // 瀑布图特有配置
    risingFill: '#5B8FF9',   // 上升柱子颜色
    fallingFill: '#E86452',  // 下降柱子颜色
    total: {
        label: '净利润',
        style: { fill: '#1D1D1F' },
    },
    
    label: {
        position: 'middle',
        style: { fill: '#fff', fontSize: 11 },
        formatter: (v) => v.value > 0 ? `+${v.value}` : v.value,
    },
    
    columnWidthRatio: 0.5,
    columnStyle: { radius: [4, 4, 0, 0] },
    
    xAxis: {
        label: { style: { fontSize: 11, fill: '#1D1D1F' } },
        line: { style: { stroke: '#E8E8ED' } },
    },
    yAxis: {
        label: { 
            style: { fontSize: 11, fill: '#1D1D1F' },
            formatter: (v) => v.toLocaleString(),
        },
        grid: { line: { style: { stroke: '#E8E8ED', lineDash: [4, 4] } } },
    },
    legend: false,
};
瀑布图规范
参数值说明
xField string 分类字段
yField string 数值字段（正负值）
risingFill #5B8FF9 上升柱子颜色（正值）
fallingFill #E86452 下降柱子颜色（负值）
total object 合计项配置（value为null时自动计算）
columnWidthRatio 0.5 柱体宽度占比
columnStyle.radius [4,4,0,0] 柱体圆角（顶部）
label.position middle 标签位置（柱体中间）
适用场景
✓ 适合
  • 财务利润分解分析
  • 成本构成展示
  • 预算差异分析
  • 数据增减变化追踪
✗ 不适合
  • 时间序列趋势展示
  • 简单数值对比
  • 占比类数据
  • 无累计关系的数据
