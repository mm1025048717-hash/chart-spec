Chart Configuration 
const config = {
    data: hierarchyData,
    innerRadius: 0.3,
    
    // 层级字段配置
    hierarchyConfig: {
        field: 'value',
    },
    
    // 颜色映射
    color: ['#5B8FF9', '#5AD8A6', '#F6BD16', '#945FB9', '#E86452'],
    
    // 交互配置
    interactions: [{ type: 'element-active' }],
    
    // 标签配置
    label: {
        layout: [{ type: 'limit-in-shape' }],
    },
    
    sunburstStyle: {
        stroke: '#fff',
        lineWidth: 1,
    },
};
旭日图规范
参数值说明
innerRadius 0.25 - 0.3 内径比例，控制中心圆大小
hierarchyConfig.field string 层级数据值字段
color array 颜色数组，支持多层级颜色映射
sunburstStyle.stroke #fff 扇形边框颜色
sunburstStyle.lineWidth 1 扇形边框宽度
label.layout [{ type: 'limit-in-shape' }] 标签布局，防止溢出
label.style.fontSize 10px 标签字号
尺寸规范
参数值说明
容器高度 400px 推荐使用大型图表高度
层级深度 2-4层 建议不超过4层，否则难以阅读
扇形最小角度 ≥5° 确保标签可读性
适用场景
✓ 适合
  • 组织架构展示
  • 产品分类层级
  • 文件目录结构
  • 地理区域层级（国家-省-市）
✗ 不适合
  • 时间序列数据
  • 非层级关系数据
  • 简单占比展示
  • 层级过深（>4层）的数据
完整代码示例
const config = { data: hierarchyData, innerRadius: 0.25, hierarchyConfig: { field: 'value', }, color: ['#5B8FF9', '#5AD8A6', '#F6BD16', '#945FB9'], label: { layout: [{ type: 'limit-in-shape' }], style: { fontSize: 10, fill: '#fff' }, }, sunburstStyle: { stroke: '#fff', lineWidth: 1, }, interactions: [{ type: 'element-active' }], };
