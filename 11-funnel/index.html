Chart Configuration 
new Funnel('chart-basic', {
    data,
    xField: 'stage',
    yField: 'value',
    padding: [24, 80, 24, 24],
    color: ['#5B8FF9', '#5AD8A6', '#5D7092', '#F6BD16', '#E86452'],
    legend: false,
    label: {
        formatter: (datum) => `${datum.stage}\n${datum.value.toLocaleString()}`,
        style: { fontSize: 11, fill: '#fff' },
    },
    conversionTag: {
        formatter: (datum) => `${((datum.$$percentage$$) * 100).toFixed(1)}%`,
    },
}).render();
漏斗图规范
参数值说明
xField string 阶段名称字段
yField string 数值字段
isTransposed false 是否横向显示（默认纵向）
dynamicHeight false 是否动态高度（按比例）
conversionTag object 转化率标签配置
label.style.fontSize 11px 标签字号
padding [24,80,24,24] 右侧留出空间给转化率标签
移动端参数配置 Mobile
参数 桌面端 移动端 说明
height 300px 220px 图表高度适配小屏
padding [24, 80, 24, 24] [16, 60, 16, 16] 减少内边距，增大图表区
label.style.fontSize 11px 10px 标签字号缩小
conversionTag.style.fontSize 12px 10px 转化率标签字号
conversionTag.offsetX 40 25 转化率标签偏移量
isTransposed false false 移动端保持纵向更节省空间
移动端优化建议
  • 阶段数量建议不超过 5 个，避免显示拥挤
  • 标签优先显示数值，阶段名称可省略或简写
  • 转化率标签使用较短格式如 "65%" 而非 "转化率: 65%"
  • 考虑使用 tooltip 显示详细信息而非直接展示
使用建议
建议说明
阶段数量建议 3-7 个阶段，过多会显得拥挤
转化率标签建议显示转化率，帮助识别瓶颈环节
颜色渐变建议使用渐变色，从深到浅表示转化递减
标签位置建议使用 inner 位置，标签清晰可见
代码示例
import { Funnel } from '@ant-design/charts'; const config = { data, xField: 'stage', yField: 'value', legend: false, label: { formatter: (datum) => `\${datum.stage}: \${datum.value}` }, conversionTag: true, };
