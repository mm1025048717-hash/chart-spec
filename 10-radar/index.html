Chart Configuration 
new Radar('chart-basic', {
    data,
    xField: 'skill',
    yField: 'score',
    seriesField: 'name',
    color: ['#5B8FF9', '#5AD8A6'],
    area: { style: { fillOpacity: 0.2 } },
    point: { size: 3 },
    padding: 'auto',
    appendPadding: [40, 20, 20, 20],
    xAxis: { label: { style: { fontSize: 11, fill: '#86868B' } } },
    yAxis: { min: 0, max: 100 },
    legend: false,

雷达图规范
参数值说明
xField string 维度字段（如技能名称）
yField string 数值字段（如评分）
seriesField string 分组字段（多人对比）
area.style.fillOpacity 0.2 填充区域透明度
point.size 3 数据点大小
yAxis.min 0 Y轴最小值
yAxis.max 100 Y轴最大值
使用建议
建议说明
维度数量建议 4-8 个维度，过少或过多都影响可读性
数值范围建议统一数值范围（如 0-100），便于对比
系列数量建议不超过 3 个系列，过多会重叠难以区分
填充透明度建议 0.15-0.3，确保多系列重叠时仍可识别
代码示例
import { Radar } from '@ant-design/charts'; const config = { data, xField: 'skill', yField: 'score', seriesField: 'name', area: { style: { fillOpacity: 0.2 } }, point: { size: 3 }, };
