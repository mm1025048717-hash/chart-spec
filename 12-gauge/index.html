仪表盘规范
参数值说明
percent number (0-1) 当前进度百分比
range.color string | string[] 进度条颜色（单色或多色分段）
range.ticks array 颜色分段点（如 [0, 1/3, 2/3, 1]）
indicator.pointer object 指针样式配置
indicator.pin object 中心点样式配置
statistic.content object 中心统计文本配置
statistic.content.style.fontSize 20-24px 中心文字字号
使用建议
建议说明
颜色分段建议使用 2-3 个颜色分段，表示不同状态（危险/警告/正常）
中心文字建议显示百分比或具体数值，字号 20-24px
指针样式建议使用浅灰色指针，不抢夺视觉焦点
适用场景适合展示单一指标的完成度、达成率、进度等
代码示例
import { Gauge } from '@ant-design/charts'; const config = { percent: 0.75, range: { color: '#5B8FF9' }, indicator: { pointer: { style: { stroke: '#D0D0D0' } }, pin: { style: { stroke: '#D0D0D0' } }, }, statistic: { content: { formatter: ({ percent }) => `达成率: \${(percent * 100).toFixed(0)}%`, }, }, };
