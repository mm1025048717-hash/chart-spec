Chart Configuration 
new Treemap('chart-basic', {
    data,
    colorField: 'name',
    padding: mobile ? [20, 20, 20, 20] : [25, 20, 25, 20],  // 平衡左右padding，让图表居中，横向图例不需要太多右侧空间
    color: ['#5B8FF9', '#5AD8A6', '#FF9845', '#FF3B30', '#5856D6', '#32ADE6', '#FF99C3'],
    label: {
        style: { fontSize: 12, fill: '#fff' },
        formatter: (v) => v.name,
    },
    rectStyle: { stroke: '#fff', lineWidth: 2 },
    interactions: [{ type: 'treemap-drill-down' }],
}).render();
矩形树图规范
参数值说明
colorField string 颜色映射字段
hierarchyConfig object 层级配置（支持嵌套数据）
rectStyle.stroke #fff 矩形边框颜色
rectStyle.lineWidth 2 矩形边框宽度
label.style.fontSize 12px 标签字号
label.style.fill #fff 标签颜色（通常为白色）
interactions [{ type: 'treemap-drill-down' }] 下钻交互配置
使用建议
建议说明
层级深度建议 2-3 层，过深难以识别
矩形大小最小矩形应足够大以显示标签，建议至少 40×40px
颜色区分建议使用不同颜色区分不同类别，增强可读性
适用场景适合展示层级数据的占比关系，如部门预算、产品分类等
