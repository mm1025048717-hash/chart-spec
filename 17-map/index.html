🗺️ 地理图表

地图 Map

在地图上展示地理位置相关的数据，支持行政区划、散点、路径等多种形式

L7Plot AntV/L7 地理可视化
地图类型
🗾
区域地图
Choropleth 行政区划填色
📍
散点地图
DotMap 点位分布
🔥
热力地图
HeatMap 密度分布
区域地图代码示例
import { Choropleth } from '@antv/l7plot'; new Choropleth('container', { map: { type: 'mapbox', style: 'light', center: [116.4, 39.9], zoom: 3, }, source: { data, joinBy: { sourceField: 'name', geoField: 'name', }, }, color: { field: 'value', value: ['#E8F3FF', '#5B8FF9'], }, });
散点地图代码示例
import { DotMap } from '@antv/l7plot'; new DotMap('container', { map: { type: 'mapbox', style: 'light' }, source: { data, parser: { type: 'json', x: 'lng', y: 'lat', }, }, color: '#5B8FF9', size: 5, });
使用场景
类型适用场景
区域地图各省销售额、GDP分布等按行政区划的数据
散点地图门店位置、事件发生地点等精确坐标数据
热力地图人口密度、订单分布等密度型数据
路径地图物流路线、迁徙路径等连线数据
⚠️ 注意事项
1. 地图可视化需要使用专门的 L7Plot 或 L7 库
2. 需要加载对应的 GeoJSON 数据
3. 地图底图服务需要 API Key（高德/Mapbox等）
