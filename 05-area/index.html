面积图规范
参数值说明
smooth true 平滑曲线（面积图常用平滑）
areaStyle.fillOpacity 0.3 填充透明度
line.style.lineWidth 2 边界线宽度
color #5AD8A6 面积填充颜色
padding [40,20,50,50] 图表内边距 [上,右,下,左]
坐标轴配置
参数X轴值Y轴值
label.fontSize11px11px
label.fill#86868B#86868B
line.stroke#E8E8EDnull
grid.linenull#E8E8ED [4,4]
使用建议
建议说明
平滑曲线面积图推荐使用平滑曲线，视觉效果更柔和
填充透明度建议 0.25-0.4，过透明难以识别，过实遮挡下方内容
堆叠面积多系列数据建议使用堆叠面积图，展示总量和构成
渐变填充单系列面积图可使用渐变填充增强视觉效果
代码示例
import { Area } from '@ant-design/charts'; const config = { data, xField: 'date', yField: 'value', smooth: true, color: '#5AD8A6', areaStyle: { fillOpacity: 0.3, }, line: { style: { lineWidth: 2 }, }, };
