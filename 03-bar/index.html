尺寸规范
参数值说明
padding [30, 20, 40, 150] 左侧留出更多空间放置长标签
容器高度 500px 条形图通常需要更高的容器
maxBarWidth 48px 条形最大厚度（像素）
minBarWidth 8px 条形最小厚度（像素）
条形规范
参数值说明
xField string 数值字段（水平方向）
yField string 分类字段（垂直方向）
barWidthRatio 0.5 条形高度占比
maxBarWidth 48 最大厚度（像素）
minBarWidth 8 最小厚度（像素）
barStyle.radius [0, 4, 4, 0] 右侧圆角（与柱状图不同）
柱状图 vs 条形图
特性柱状图 Column条形图 Bar
方向 垂直 水平
标签空间 X轴空间有限 Y轴空间充足
适用标签 短标签（≤8字符） 长标签（产品名等）
圆角方向 [4,4,0,0] 顶部 [0,4,4,0] 右侧
完整代码示例
const config = { data, xField: 'sales', // 数值字段（水平） yField: 'product', // 分类字段（垂直） padding: [30, 20, 40, 150], color: '#5B8FF9', barWidthRatio: 0.5, maxBarWidth: 48, // 最大厚度 minBarWidth: 8, // 最小厚度 barStyle: { radius: [0, 4, 4, 0], // 右侧圆角 }, label: { position: 'right', offset: 8, style: { fontSize: 11, fill: '#5B8FF9' }, }, xAxis: { label: { formatter: (v) => (v / 1000).toFixed(0) + 'k' }, grid: { line: { style: { stroke: '#E8E8ED', lineDash: [4, 4] } } }, }, yAxis: { label: { style: { fontSize: 11, fill: '#86868B' } }, }, };
