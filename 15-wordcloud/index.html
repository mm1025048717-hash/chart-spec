    wordField: 'word',
    weightField: 'weight',

词云图规范
参数值说明
wordField string 文字内容字段
weightField string 权重字段（决定大小）
colorField string 颜色映射字段
wordStyle.fontSize [14, 48] 字体大小范围 [最小, 最大]
wordStyle.rotation 0 文字旋转角度（0为不旋转）
color array 颜色数组，支持多色映射
使用建议
建议说明
词数量建议 20-100 个词，过少效果不明显，过多难以阅读
字体大小建议范围 [12, 48]，确保重要词汇突出
旋转角度建议 0 或 [-90, 90]，避免过度旋转影响阅读
适用场景适合展示文本数据的关键词频率，如用户评论、搜索热词等
代码示例
import { WordCloud } from '@ant-design/charts'; const config = { data, wordField: 'word', weightField: 'weight', colorField: 'word', wordStyle: { fontSize: [12, 48], rotation: 0, }, };
