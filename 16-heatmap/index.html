Chart Configuration 
new Heatmap('chart-basic', {
    data,
    xField: 'hour',
    yField: 'day',
    colorField: 'value',
    color: ['#E8F3FF', '#5B8FF9', '#0051D4'],
    shape: 'rect',
    xAxis: {
        label: {
            style: { fontSize: 11, fill: '#86868B' },
            formatter: (v) => v.split(':')[0],
        },
    },
    yAxis: {
        label: { style: { fontSize: 11 } },
    },
    legend: { position: 'bottom' },
}).render();
热力图规范
参数值说明
xField string X轴字段（分类或时间）
yField string Y轴字段（分类）
colorField string 颜色映射字段（数值）
color string[] 颜色渐变数组（顺序色板）
shape rect 热力单元形状（矩形）
legend.position bottom 图例位置（底部）
使用建议
建议说明
颜色映射建议使用顺序色板，从浅到深表示数值大小
数据密度适合展示二维分类数据的数值分布，如时间×类别的数据
适用场景适合展示访问量热力图、销售热力图、相关性矩阵等
代码示例
import { Heatmap } from '@ant-design/charts'; const config = { data, xField: 'hour', yField: 'day', colorField: 'value', color: ['#F0F0F0', '#5B8FF9'], // 浅到深 shape: 'rect', };
