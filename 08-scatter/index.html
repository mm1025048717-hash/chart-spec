散点图规范
参数值说明
xField string X轴字段（数值型）
yField string Y轴字段（数值型）
colorField string 颜色分组字段（可选）
size 5 点的大小（固定大小）
shape circle 点的形状（circle、square等）
pointStyle.fillOpacity 0.8 点的填充透明度
坐标轴配置
参数X轴值Y轴值
label.fontSize11px11px
label.fill#1D1D1F#1D1D1F
grid.line#E8E8ED [4,4]#E8E8ED [4,4]
使用建议
建议说明
数据量适合展示 50-500 个数据点，过多会重叠难以识别
点的大小建议 4-6px，确保不重叠且清晰可见
颜色分组建议使用 colorField 区分不同类别，增强可读性
适用场景适合展示两个连续变量之间的关系，如相关性分析
代码示例
import { Scatter } from '@ant-design/charts'; const config = { data, xField: 'advertising', yField: 'sales', colorField: 'category', size: 5, shape: 'circle', pointStyle: { fillOpacity: 0.8 }, };
